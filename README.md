# Win-Fi v0.9-Alpha

This is a Python3 script which shows saved WiFi Passwords in Windows based on **netsh**.

## Features:
- Shows Saved Passwords
- Shows Network Type
- Shows Version
- Shows Security Type

## Requirements:
- Windows OS
- Python3
- netsh

## How to use?
Open Command Prompt & Type this command:
```
python start.py
```
